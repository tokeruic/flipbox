## ③モーターモジュールからハンドルモジュールを外す。
**紙をセットする際や、ハンドルを回してアニメーションさせる場合は、必ずモーターモジュールからハンドルモジュールを取り外してください。**  
**歯車が噛み合っている時に無理やりハンドルを回すと故障の原因になります。**

プラス・マイナスにお気をつけください。バネがある方がマイナスです。
